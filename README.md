
# KYK Yaşam Koçu

Bu proje, KYK yurtlarında kalan öğrenciler için geliştirilmiş bir uygulamadır.

## Özellikler
- Yemek listesi görüntüleme
- Yemeklere puan ve yorum yapma
- Çamaşırhanedeki boş makineleri görüntüleme
- Duyuru sistemi (sadece admin paylaşabilir)
- Kullanıcı giriş sistemi (C# + SQL)

## Kullanılan Teknolojiler
- C#
- SQL Server
- Windows Forms

## Amaç
Öğrencilerin yurt yaşamını kolaylaştırmak ve geri bildirim sistemini dijitalleştirmek.
